1. Countdown Timer

A Python countdown timer built as part of my Python learning journey.

2. Description

This program takes a time value from the user and counts down in real time until it reaches one.

3. Features

- Takes time input from the user
- Updates every second
- Uses Python's time module
- Stops automatically when the countdown reaches one

4. Technologies Used

- Python
- time module

5. Concepts Practiced

- Loops
- User Input
- Variables
- Time Handling
- Program Flow Control

6. What I Learned

- How to use the time module
- How countdown timers work
- How to update output continuously inside a loop
